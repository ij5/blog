리액트 중첩된 함수 안에서 state 업데이트

October 29, 2021 · 1 min · Jaehee Lee | Suggest Changes

React의 함수형 컴포넌트에서 중첩된 함수 내 list state를 업데이트 할 경우 데이터가 제대로 업데이트가 되지 않는 문제가 발생한다.

해결 방법:

setListState([...listState, data])

를

setListState(currentListState => [...currentListState, data]);

로 바꿔줘야 한다.

이거 찾느라 몇시간 고생했다.
